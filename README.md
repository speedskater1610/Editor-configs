My Neovim config was the first editor, that I used, after I started program. Well not the first editor, that was VS-code, which I used the default config for. after about 6 months of programming, I swicthed to Manjaro linux, where I wanted a lighter weight editor, since my 2011 macbook air couldn't handle the so called "lightweight" editor, VS-code. So I started looking for a new editor, in this time I tried around 4 diffrent truly lightweight editors, My favorite of the them, was neovim. So I switched and used LazyVimi while I learned Lua, so that I could, then write my own config. Once I learned Lua I made a new config and started to use a custom neovim config, although this was a very diffrent editor than it is in this repo, I used this editor for about 4 months before I entered my purest fase, where I started to write almost only rust, and haskell, so I switch to vim as my main editor.


Once I entered thsi purest face, we all go through one right?, I started to use Vim and Arch linux, because I was to good for Neovim, I learned Vim Script in about a week and in that time only uses nano to write my config, See I was a true purest...
